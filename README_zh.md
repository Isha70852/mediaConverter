# MediaConverter

> **瀏覽器端 AI 影像後製工具 — 去除浮水印 · 影片轉檔 · 壓縮 · 分割**
> ⚠️ 目前為 Beta 測試版本，歡迎回報問題與建議

## 簡介

MediaConverter 是一款完全在瀏覽器中執行的影像處理工具，無需安裝任何程式或上傳至伺服器。  
主要針對 AI 生成內容（如 nano banana 圖片、Gemini Omni Flash 影片）的浮水印去除需求設計，並整合多項實用的影片處理功能。

## 功能

### 1. 去除浮水印

移除 AI 工具生成內容右下角的浮水印：

- 支援 **nano banana** 生成圖片的浮水印去除
- 支援 **Gemini Omni Flash** 生成影片的浮水印去除
- 可自訂浮水印遮蓋區塊的**位置**與**大小**，以精確對齊原始浮水印
- 💡 **建議**：處理影片時，將 `Alpha Scale` 設為 `0.6` 可獲得最佳效果

### 2. 影片轉檔

- 支援大多數主流影片格式轉換為 **MP4**

### 3. 影片壓縮

- 透過調整**位元率（Bitrate）**與**解析度（Resolution）**，有效降低影片檔案大小

### 4. 影片分割

- 自訂起始與結束時間，裁剪保留影片中所需的片段

## 技術架構

| 元件 | 說明 |
|------|------|
| [mediabunny](https://github.com/Vanilagy/mediabunny) | 主要影片處理引擎，支援常見格式 |
| [ffmpeg.wasm](https://github.com/ffmpegwasm/ffmpeg.wasm) | 備用處理引擎，用於 mediabunny 不支援的格式 |

特別感謝 **[Allen Kuo](https://github.com/allenk/GeminiWatermarkTool)** 提供去除浮水印的核心功能構想。

## 已知限制

- 部分影片或圖片的浮水印可能無法完全去除（持續優化中）
